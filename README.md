# Project Title

Submission Bookshelf API

## Requirements

For development, you will only need Node.js >= v.18.16.0 and a node global package, Npm, installed in your environement.

### Set-up and Run
> now install dependencies and run the project

```shell
$ npm install
$ npm run start
```
